## PT2025_30222_Flita_Andrei_1 - Task Management System
## Description
This Task Management System allows users to organize and manage tasks in a hierarchical manner. Tasks can be complex and have subtasks, and the system displays them across multiple lists in a graphical user interface (GUI). The application enables users to:

->Add/View employees

->Add/View tasks(simple or complex)

->Assign tasks to employees

->Change the status of a task(completed or uncompleted)

->Calculate how many hours an employee has worked based on the number of completed tasks and their duration

## Installation
To get started with the Task Management System, follow the steps below.

1.Clone the repository to your local machine.

2.Run the project.

3.Start using the system.
## Usage
Once the application is running, you can interact with it as follows:

1.Adding Employees , the first button you are going to see is just for adding an employee.

2.Adding Tasks, the second button will allow you to add a task simple or complex and for the complex task give you the possibility to give it other subtasks.

3.Assign Tasks, the third button allows you to assign a task to an employee.

4.Show Employees, the fourth button allows you to see the list of employees with their assigned tasks. It also allows you to change the status of a task or calculate the work duration of the employee.

5.Show Tasks, the fifth and final button allows you to see the tasks you created along with the subtasks for the complex tasks.
